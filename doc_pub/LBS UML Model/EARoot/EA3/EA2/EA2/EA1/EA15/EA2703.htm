<p>Because LBS can support multiple requests of different types (e.g. MTLR+MOLR+X3P+CellMOLR) at the same time. All the request quality parameters must be merged and a proper positioning mode(s) must be choosen.<br/></p><p><br/></p><p>Due to the fact that all AGPS positioning sessions share the same AGPS integration module there is a possibility of the new location request being rejected.<br/></p><p><br/></p><p>1. A positioning session (its location request) may fail if:<br/></p><p>   a) AGPS Module does not support simultaneous Terminal Based and Terminal Assisted positioning, and<br/></p><p>   b) AGPS Module is busy processing a location request in one of the modes, and<br/></p><p>   c) a new positioning session (its location request) requires a position to be calculated in another mode<br/></p><p><br/></p><p>2. A positioning session (its location request) may fail if the AGPS module does not support the requested positioning mode at all.<br/></p><p><br/></p><p>The failure scenarios are uncommon and for simplicity are not shown in this sequence.<br/></p>
