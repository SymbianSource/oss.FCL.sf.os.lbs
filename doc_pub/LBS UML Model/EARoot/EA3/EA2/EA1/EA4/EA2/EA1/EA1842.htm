<p>The Privacy Request API implements a subset of the functionality of the Protocol Module API. Because the two interfaces share the same concept of privacy validation, the Privacy Protocol Module apart from providing the Client/Server functionality does require any specific logic to couple the two interfaces. Call mapping is straight forward.<br/></p><p><br/></p><p>All the interactions within the LBS sub-system as well as between LBS and an appropriate privacy validation application (e.g. Privacy Controller) are independent on the type of the interface used for a privacy request injection (Network Privacy API or Privacy Request API).<br/></p><p>For details see the sequence diagrams presenting the usage of the Network Privacy API.<br/></p>
